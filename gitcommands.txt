git clone {REPOSITORY LINK} (THIS COMMAND IS USED TO GET REPOSTORY FROM GITHUB)
git status (TO CHECK GIT FOLDER STATUS)
cd {filename} (USED TO ENTER INTO THE FOLDER)
git add [local file name in a folder which does not exists in github} (USED TO MOVE LOCALfile in FOLDER INTO GITHUB FOLDER)
git add . (this command is used to add all files in folder )
git commit -m "message"(THIS IS USED TO DECLARE MESSAGE WHICH IS VERY IMPORTANT IN GITHUB)
git push origin main (THIS COMMAND MOVES LOCAL FOLDER TO GITHUB FOLDER)
git pull (command used to get file which created in github)
git rm (this command used to delete files in github)
cd .. (command used come out from old file or directory)
git init (this command is used to make normal file to git file)
git branch (this command is used to our branch name of the file)
git branch -m "new branch name you want" (this command is used to change the default branch name)
git remote add origin (this command is used to add new file to github repository)
git branch -a (this command is used to see currently which branch we are)
git checkout (this command used to switch one branch to another)
git diff {branch name} (this command is used to know differnce between branches)
git merge {branch name which you want to merge} (this command is used to merge 2 branches into 1)
